# Flip and Flop – Memory Card Game

A simple and elegant memory card matching game built using **HTML, CSS, and JavaScript**.

**Live Demo:**  
https://priyadharshinirvp.github.io/flip-flop/

---

## About The Project

Flip and Flop is a browser-based memory game where players flip cards to find matching pairs.  
The game tracks the number of moves and time taken, and supports multiple difficulty levels.

This project focuses on:
- DOM manipulation
- Event handling
- Game logic implementation
- Responsive UI design
- Clean and modern styling

---

## Features

- Multiple difficulty levels (Easy, Medium, Hard)
- Live timer
- Move counter
- Elegant and modern UI
- Dynamic card generation
- Game reset functionality
- Fully hosted using GitHub Pages

---

##  Built With

- **HTML5** – Structure
- **CSS3** – Styling & layout
- **JavaScript (ES6)** – Game logic & interactivity

---

## How to Play

1. Select a difficulty level.
2. Click **Start Game**.
3. Flip two cards at a time.
4. Match all pairs to complete the level.
5. Try to finish in fewer moves and less time!
